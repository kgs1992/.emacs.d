### Installation:
emacs-plus:
```
brew install emacs-plus@28 --with-native-comp
```
Clone & install:
```
git clone git@github.com:kgs1992/.emacs.d.git ~/.emacs.d
cd ~/.emacs.d
make
```

### Update packages:
```
cd ~/.emacs.d
make update
```
